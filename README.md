# Jackson Hole 2026 Trip Coordinator

A ski trip coordination app for group trips.

## URLs

- `/?admin=true` - Admin view (manage people, housing, etc.)
- `/?p=slug` - Person's individual view

## Setup

```bash
npm install
npm start
```

## Deploy to Vercel

1. Push to GitHub
2. Import project on vercel.com
3. Deploy automatically

## Features

- Flight tracking with status links
- Housing assignments with access codes
- Mountain info (webcams, trail maps, hours)
- Local spots & events
- Meetup coordination
- Grocery list
